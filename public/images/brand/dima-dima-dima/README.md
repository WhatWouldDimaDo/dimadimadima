# Dima Dima Dima Mark

Selected direction: concept 20 from `brand-exploration/2026-07-14/refinement-03`.

The artwork is rebuilt as deterministic SVG geometry rather than upscaling the generated raster. The profile is a true transparent cutout; none of the SVGs contains a background rectangle.

## Masters

- `dima-mark-color.svg` — dark teal D and copper beard for light backgrounds.
- `dima-mark-color-dark.svg` — warm white D and brighter copper beard for dark backgrounds.
- `dima-mark-mono.svg` — single-color dark teal.
- `dima-mark-mono-reverse.svg` — single-color warm white.

The `png/` folder contains transparent exports at 1024, 512, 192, 64, and 32 px. `dima-mark-usage-proof.png` checks the four color modes; `dima-mark-size-proof.png` checks legibility from 16–128 px.

## Palette

- Deep teal: `#0B5A58`
- Copper beard: `#C95316`
- Dark-mode copper: `#D8692B`
- Warm white: `#F8F1E5`

The existing On Dima's Radar face/radar mark remains the project-specific identity. This D mark is the umbrella identity for Dima Dima Dima.
